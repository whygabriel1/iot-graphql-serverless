
This repository demonstrates a clean and scalable implementation of a GraphQL API designed for IoT device monitoring. 

## 🛠 Tech Stack
- **Runtime:** Node.js
- **Language:** TypeScript
- **API Architecture:** GraphQL (Apollo Server)
- **Patterns:** Serverless, Repository Pattern, SOLID Principles

## 📝 Features
- Real-time device status querying.
- Scalable schema for high-density IoT environments.
- Optimized for AWS Lambda deployment.

*This project was built to showcase modern backend architecture and rapid prototyping capabilities.*
